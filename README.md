# **The classic Minesweeper game using *`java`***

 available in *English* and *German*
 
 If you haven't already done: download *[processing](https://processing.org/download/)*
 

**Controls**

|input|action|
|---|---|
|[leftMouse]|reveal tile|
|[rightMouse]|set flag/remove flag|
